Algorithms and Data Structures

This is a collection of C++ projects that use object-oriented classes to pass a test file using the correct functions and/or solving a puzzle.

Projects:

  • Maze Solver
  • Data Structure Classes

Missionaries and Cannibals Problem

In the Missionaries and Cannibals problem, there are 3 missionaries and 3 cannibals on one side of a river. They all need to cross the river in a boat that can take two at a time. The puzzle is that on neither side of the river can their be more cannibals than missionaries at one time.

The MazeSolver project recursively searches a multidimensional array looking for open spaces where the path through the maze can continue until it reaches the exit.

In this program, it is tested with inserting values into an array after they have been hashed. If two values end up being assigned the same index, the class uses quadratic probing recursively until it finds an empty index to store the value

The Stack data structure class creates Nodes. These nodes hold their respective data and a pointer to the next Node. The Stack constructor only holds the top, and bottom nodes, as well as the size. With the stack you are able to push a new node which will add to the top of the stack and pop to remove the item at the top of the stack as well as get the popped node returned. The class also has the ability to print out the entire stack to the console. As well as some other helper functions.
