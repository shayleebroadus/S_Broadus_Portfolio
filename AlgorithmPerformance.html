Algorithms and Data Structures in C++

This is a collection of C++ projects that use object-oriented classes to pass a test file using the correct functions and/or solving a puzzle.

Projects:

  • Missionaries and Cannibals Puzzle
  • Maze Solver
  • Data Structure Classes

Missionaries and Cannibals Problem

In the Missionaries and Cannibals problem, there are 3 missionaries and 3 cannibals on one side of a river. They all need to cross the river in a boat that can take two at a time. The puzzle is that on neither side of the river can their be more cannibals than missionaries at one time.

The MazeSolver project recursively searches a multidimensional array looking for open spaces where the path through the maze can continue until it reaches the exit.

In this program, it is tested with inserting values into an array after they have been hashed. If two values end up being assigned the same index, the class uses the quadratic probing equation recursively until it finds an empty index to store the value. This class has fairly typical functions for a data structure class. The ability to create a hashtable in many ways or copy an existing one. The object can be checked for its size, capacity, if it is empty. The class can add, remove, search for specific items, or resize itself once all of the available slots have been used.

This program implements and tests a class of Node object that are all linked into a Queue object. Each node object has its data of any type, and a pointer to the next item in the queue. The class is equipped with functions to see the state of the queue such as if it is empty or the count of how many objects are inside. Also has the function "peek" which returns the data from the node at the top but does not remove it from the queue. Then the standard add, remove, and print functions to more directly interact with the data.

The Stack data structure class creates Nodes. These nodes hold their respective data and a pointer to the next Node. The Stack constructor only holds the top, and bottom nodes, as well as the size. With the stack you are able to push a new node which will add to the top of the stack and pop to remove the item at the top of the stack as well as get the popped node returned. The class also has the ability to print out the entire stack to the console. As well as some other helper functions.
